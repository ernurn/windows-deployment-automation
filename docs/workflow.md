\# Workflow



\## Overview



This document describes the standard workflow for Windows deployment using cloning and post-automation scripts.



\---



\## Deployment Process



1\. Prepare the Windows image  

2\. Capture the image using Clonezilla  

3\. Deploy the image to target machines  

4\. Run post-clone script  

5\. System is ready for delivery  



\---



\## Goal



\- Reduce manual configuration

\- Ensure consistency across devices

\- Speed up deployment process
