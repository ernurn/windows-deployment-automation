\# Naming Convention



\## Overview



This project uses an automatic naming system to assign unique hostnames after cloning.



\---



\## Format



PC-XXXXX



\---



\## Logic



1\. Try to retrieve BIOS Serial Number  

2\. If invalid → fallback to MAC Address  

3\. Extract last digits as unique identifier  



\---



\## Example



PC-A1B2C3



\---



\## Benefits



\- Avoid duplicate hostnames

\- Ensure uniqueness in network environments

\- Fully automated process
