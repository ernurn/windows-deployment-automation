# Future Improvements

This document outlines planned enhancements to evolve the project towards a more scalable, automated and enterprise-ready deployment system.

---

## 🚀 Automation
- Dynamic GRUB menu generation (auto-detect available images)
- Parameterized deployments (disk selection, modes)
- Pre and post-deployment hooks for custom automation
- Script modularization (reusable components)

---

## 🌐 Infrastructure
- Network-based deployment (PXE boot)
- Centralized image repository
- Remote configuration capabilities
- Multi-site deployment support

---

## 📊 Management
- Asset management integration
- Deployment tracking system
- Hostname-to-device mapping database
- Basic inventory system

---

## 📊 Observability

- Centralized logging (log aggregation)
- Remote log collection
- Deployment success/failure tracking
- Basic alerting system

---

## 🧠 Advanced Features
- Environment-specific configurations (RMA, lab, production)
- Custom hostname prefixes per environment
- Role-based deployment profiles
- Integration with configuration management tools

---

## 🔐 Security

- Script execution hardening
- Secure credential handling
- Integrity validation of deployment images
